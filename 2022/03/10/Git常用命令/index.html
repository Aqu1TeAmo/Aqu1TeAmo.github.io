Git常用命令

发表于 2022-03-10 分类于 技术学习

仓库相关命令

git init 初始化一个仓库，将文件夹初始化成git仓库
git add 将文件提交到git暂存区 （git add -A 提交所有文件到git暂存区）
git commit -m “提示信息” 将文件提交到git仓库 “提示信息”通常用来描述此次提交所做更改
git commit -m “first commit” 提示这是第一次提交
git log –stat 查看提交的历史记录
git checkout 在未提交的情况下，撤回对文件的更改
git reset HEAD^1 在已经提交的情况下，撤回对文件上次的更改

分支相关命令

git checkout -b 以当前分支为基础创建新的分支
git checkcout 切换分支
git brance 列举所有分支
git brance -D 删除分支
git merge 合并分支
git merge –abort 放弃分支合并
git push 推送当前分支最新的提交到远程
git pull 拉取远程分支最新的提交到本地
